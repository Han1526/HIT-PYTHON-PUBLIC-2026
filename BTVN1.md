Bài 1: Python là ngôn ngữ thông dịch hay biên dịch? Tại sao?
-Trả lời: Python là ngôn ngữ thông dịch (interpreted language), nhưng bên trong vẫn có bước biên dịch sang bytecode trước khi thực thi.Vì chương trình được thực thi thông qua Python Interpreter, chứ không tạo ra file thực thi (.exe) độc lập như C/C++.
Bài 2:  Liệt kê:
-Trả lời:
+Các kiểu dữ liệu trong Python: int, float, complex, str, bool, list, tuple, set, dict, None
+Các toán tử trong Python: +; -; *; /; //; %; **; ==; !=; >; <; >=; <=; =; +=; -=; *=; /=; %=; **=; is; is not; and; or; not
+Mệnh đề điều kiện và vòng lặp là if...elif...else; vòng lặp for, while
+Kiểu dữ liệu True, False
